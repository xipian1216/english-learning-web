## 目的

展示学习者的词汇本。当前页面是未来词汇管理功能的静态界面壳。

## 当前行为

- 根据 `wordsCount` 显示总词汇数。
- 根据 `lastWord` 显示最后添加的词汇。
- 渲染静态示例词汇卡片。
- 使用响应式网格布局。

## 当前限制

- 不会从 API 加载数据。
- 尚未使用 Pinia 词汇 Store。
- Header 中的搜索输入框不会影响页面内容。
- 尚未实现新增、编辑或删除交互。
