# 词汇管理

## 状态

计划中。

## 摘要

词汇本应从静态界面壳演进为可用的英语词汇管理功能。

## 当前基线

`VocabularyNotebook.vue` 当前渲染静态示例词汇和统计卡片。`Header.vue` 包含视觉上的搜索输入框，但尚未实现搜索行为。

## 潜在范围

- 从后端 API 加载词汇条目。
- 新增词汇。
- 编辑词汇、笔记、标签或学习元数据。
- 删除或归档词汇。
- 搜索和过滤词汇。
- 基于真实数据统计总词汇数和最后添加词汇。
- 增加加载、空状态和错误状态。

## 待确认问题

- 词汇条目应包含哪些字段？
- 搜索应在本地、服务端还是两者结合完成？
- 词汇状态是否应放入 Pinia Store？
- 第一阶段实现应由哪些测试覆盖？
