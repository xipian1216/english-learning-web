# 当前版本

## 版本

`0.0.0`

## 状态

早期开发阶段。

## 已包含能力

- Vue 3 + Vite 应用壳。
- 基于路由的布局。
- 登录、注册和密码重置认证路由。
- 受保护的词汇本路由。
- 使用 `localStorage` 持久化的 Pinia auth store。
- 面向注册、登录、当前用户和密码更新的认证 API 客户端。
- 使用 Vitest 和 Playwright 的初始测试配置。

## 已知限制

- 密码重置后端集成尚未完成。
- 词汇本使用静态示例数据。
- 认证 UI 错误处理尚不完整。
- 测试需要更强的 Mock 和覆盖率。
